# Frontend Mentor - Recipe Page Solution

Bu proje, [Frontend Mentor](https://www.frontendmentor.io) üzerindeki "Recipe page" meydan okumasının bir çözümüdür.

## Özellikler
- HTML5 semantik etiketleri kullanıldı.
- CSS Flexbox ve tablo yapıları uygulandı.
- Özel liste işaretçileri (`::marker`) kullanıldı.
- Responsive tasarım mantığıyla oluşturuldu.

## Kullanılan Teknolojiler
- HTML
- CSS (Custom properties, Pseudo-elements)
